北京五日游

去看看> 不再显示
  • 出发时间/2017-11-01
  • 出行天数/5 天
  • 人物/家庭出游
  • 人均费用/2000RMB

旅行时间：2017.11.01～2017.11.05
主要行程安排：
第一天，天坛公园
第二天，天安门→王府井→故宫
第三天，恭王府→圆明园→南锣鼓巷
第四天，八达岭长城
第五天，前门大街闲逛



第一天，天坛公园
上午9点半的航班，上海到北京，大概11点半到北京首都机场，办了交通卡，坐机场线直达东直门地铁站，25元，然后2号线到前门站，c出口直接能看到毛主席纪念馆，所以住宿在前门站，每条胡同都有很多住宿的地方，基本都是经济型，携程上订就可以了。然后12点半多到达酒店，放好东西，去四季明福炸酱面吃了炸酱面，然后出发去天坛公园，公交车忘了几路，自己百度地图就可以了。然后天坛公园推荐套票，28。因为我是分开买的，30元。大家能省就省咯，哈哈哈。
下面发几张自己的美图

晚上吃了，四季明福烤鸭店。推荐点一只鸭，半只不合算。

第二天：天安门→王府井→故宫
为了看升旗，六点多就出发了，一出胡同就有人卖小红旗，买了两个，街边好多店也开了，跟着路人走就可以，走到地铁站那，就可以看到好多旅游团，跟着他们走就可以，很多人问去不去天安门，其实很近的，坐不坐看你们自己了。

能看到，一路上开了好多家店了。

人很多的，我看不到，最后升完旗，才挤到前面拍了个照片，兵哥哥都长得挺好的。

接着是去看毛爷爷，要寄包，不允许带包的，绕着毛主席纪念馆绕了好久才进去，还买了菊花，给毛爷爷很认真的鞠了三个躬，然后进去看了毛爷爷。

然后中午去的王府井，找吃的，王府井小吃街11点才营业，所以随便找了家，孔乙己，不太推荐。然后下午重新坐地铁到天安门，去故宫。故宫真的大，故宫逛完，还有景山。所以去天安门故宫不建议中午出去吃，带点吃的就行，不然太累。我真的是硬爬的景山，但是觉得爬得值得，因为可以在景山上可以看到故宫的全景，这感觉很棒。

第三天，恭王府→圆明园→南锣鼓巷
早上吃了狗不理包子，一笼58。
然后出发去恭王府，40元。大部分都是文字表述和一些文物的展示。

中午随便路边买了点小吃，就出发去圆明园了。除了长城，其他景点都推荐地铁出行。

晚上逛了南锣鼓巷，买了很多小吃。

第四天，八达岭长城
早上吃了庆丰包子，
去德胜门坐877直达八达岭，先做地铁到小积潭地铁站，然后走路到达877公交车站。人很多，但车也多，每个人都会有位置，所以也还好，就是做公交车是累，一个半多小时。
我们选的滑车，135一个人。精华段自己爬，主要就是好汉坡。
记得带吃的，然后八达岭人是多，所以下次再来北京，打算爬慕田峪，相比较八达岭人少。但是没去过长城，还是推荐八达岭吧，因为当时犹豫了很久，最终还是选了八达岭，慕田峪太远，交通不是很方便，要包车，所以下次来北京再考虑慕田峪了。在这里订个约定，还要和男朋友去北京

一包泡芙登长城，哈哈哈。

晚上去前门大街那，吃了局气，看的大众点评，感觉挺好，吃下来感觉也不错。

第五天，就是逛了逛前门大街。喝了心心念念的豆汁，身无可恋。

最后的话，因为第一次写游记，大家见谅。估计废话多了，重点也不多，但是应该也可以给大家一点参考吧。
能坐地铁就坐地铁。然后豆汁作为南方人我是真的喝不了，喝了一口没敢喝第二口，身无可恋的味道。

最后的最后，交通卡留作纪念，没退。

本篇游记共含1312个文字，67张图片。帮助了名游客。 举报
