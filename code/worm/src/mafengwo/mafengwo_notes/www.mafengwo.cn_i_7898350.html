出发前的牢骚

北京去过很多次，都是去出差，去市内的景点，去坐飞机，去倒火车～怀柔还是第一次来。

早就听说过红螺寺，别人为了求子求姻缘，我们就是奔着红叶和银杏来的～每年到了秋天，不同的攻略上就会写着不同的红螺寺。有人说，小众银杏赏秋去处；有人说，小众红叶赏秋去处～好吧，我是既想看红叶又想看银杏。
所以标题也是恶搞了一下，将两首诗前后句拼凑了起来~~

出发前，有一场考试，所以不得不将红螺寺赏秋的行程压后再压后，以至于错过了最盛大的红🍁最灿烂的黄🍂

天津到北京一般为了避免堵车，我最爱选择的还是城际再倒地铁再。。。。这次倒来倒去实在受不了，但没车怎么破？
包车太贵还有限制，最终选择了神州租车，租个大滴滴🚘🚘开往北京的秋天～

田师傅这次比较辛苦，又要选择路线，还要担任司机这一重要职务～
没有工资，还要饿肚子～很惨的样子～(捂脸小蜂)(捂脸小蜂)

第一次自驾，出发~

我们6点半集合启程，根据百度地图的导航，10点半到达红螺寺。中途办了进京证，过了一些必要的检查，8点半左右在北京六环有些堵车，田师傅也为了熟悉车况，开得慢了一些。
回来时用了2个半小时就顺利到市内了。
