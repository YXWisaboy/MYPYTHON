关于在北京吃什么，我有话要说（来自一个四川人，持续更新）

网罗北京所有好吃的

一名合格吃货的修养，就是要网罗所有好吃的，每去一个地方直接做好功课吃有准备的饭！然而...作为一个中国胃的四川鹅，看着满篇的麻酱、勾芡、浓稠咸甜的推荐菜单实在是无从下口，烤鸭虽好，也不能天天吃啊！

所以只能亲自下场，为大家提供一个LIST。如果你在尝试了爆肚、豆汁儿、麻酱、驴打滚、炸酱面之后，表示咦~~~~额.....，也许试试这个非主流的推荐也不错！

注意：这里所说的好吃，非常主观，非常主观，非常主观。重要的事情说三遍！

本人身处在被誉为美食沙漠的海淀区，每吃一次好吃的都得长途跋涉，所以更新略慢，大家见谅。

所有的推荐都是亲测过的，排名不分先后。如果真的是美味暴击会标出来滴~也希望同爱吃的蜂蜂给我推荐好吃的店，比心！

工薪消费，童叟无欺。毕竟北漂，基尼指数也不能太高了啊。

真心的求助

北京哪里有好吃的鱼火锅！！！！

北京哪里有好吃的鱼火锅！！！！

最好是河鲜类的小鱼比如：三角峰、黄辣丁、仔鲶之类的，
实在不行，梭边鱼，石斑鱼、清江鱼，花鲢、白鲢都可以啊！

龙利鱼和巴沙鱼系列请退散！不约不约~

一、夺不走的本命串串香

这个系列是我真爱，励志和串串小姐妹吃遍北京所有的串串香。因为人少也可以吃很多种菜，满足感100分！而且在我心里，串串比火锅更江湖气。一桶签签，三五瓶啤酒，就算只有一人，也不显孤单，要是有两三好友，在充满着辛辣气的烟雾氤氲中大汗淋漓，大声地讲话，就算把腿踩到椅子上得放肆也没人管得着。像是汪曾祺先生写的栀子花：栀子花粗粗大大，又香得掸都掸不开，于是为文雅人不取，以为品格不高。栀子花说：＂去你妈的，我就是要这样香，香得痛痛快快，你们他妈的管得着吗！＂

当然，事实上，要做到这么释放天性是越来越难了，可能只有坐在重庆的马路牙子上才能有这种气魄。

说回北京，如果用川渝的标准来要求，实在就有点太苛刻了。

所以只能写在我不断尝试的途中，剔除踩雷的部分（真的挺容易踩雷的，特别是美食号推荐的广告店和网红店！！还我血汗钱！！），剩下的白名单（会精确到店，每一家我都有真的吃过！），供大家解馋。而且在我的标准里，环境太小家子气的不行，少了魂魄，是做不出真正的好味道，因为小燕子是当不了还珠格格的~（这个比喻.....暴露了年龄。）

串串在手，天下我有！排名不分先后~

1、【六婆串串香火锅（圣熙八号店）】
地址：学清路甲8号商业广场2号楼一层6号-2
人均：90元/人

六婆串串香火锅

# 推荐理由：是一家我反反复复去吃好多遍的店。在成都的时候我完全不把六婆放眼里好吗？不过到了北京，连锁店的优势就体现了，品控做的好，味道不至于跑偏太多，而这家我觉得是不低于成都店的水准，还有干碟蘸料提供，别家少有。干碟蘸料是亮点！干碟蘸料是亮点！干碟蘸料是亮点！重要的事要强调！
这周五又要去啦，想想就开心！


2、【乐山飘香串串香（鼓楼店）】
地址：鼓楼东大街小经厂（近南锣鼓巷）
人均：90元/人

乐山飘香串串香

# 推荐理由：虽然只去过一次，但是记忆太深刻了！在找店的途中买了半个西瓜，到店让老板帮忙切了，一口串儿，一口西瓜的感觉真的太爽了。特别是当好多隔壁桌问老板是不能也来一份西瓜，老板摊手说：店里没有，是客人自己带过来的，然后留下一圈惺惺眼神的时候。简直是全店最酷炫好吗？

回想起来过程太美好了，又辣又爽，肾上腺素飙升，当然结局是.....吃完以后拉肚子了。。。


3、【壹零壹陆串串小吃文化馆（交道口店）】
地址：安定门内大街320号
人均：81元/人

# 推荐理由：是目前我吃过最好的干拌串串，嗯~怎么形容呢？油清而不腻，味香而不燥，在串儿上铺了满满一层的芹菜粒、洋葱粒、香菜碎和榨菜碎，拌一拌入口，混着各种香味，而且吃完身上不会有味道，相对小清新一点。可以穿着大衣踩着高跟儿去吃，吃完去蹦迪都没有问题！

二、我吃火锅，不吃底料

虽说都是吃货，但串串党和火锅党还是会为谁更优秀而打起来，为捍卫本命的名誉而战。与普通的火锅相比，我更喜欢串串，因为串串能在人少的时候获得吃很多种类的菜品的满足感。但如果说到另一类火锅，恐怕串串就没法比了，就是有主料的那种！比如芋儿鸡、烧鸡公、肥肠鸡、猪肚鸡、老鸭汤、梭边鱼、耗儿鱼、美蛙鱼头、跳水兔.....

我想相声报菜名如果报的是这些，估计要吞好多次口水才能说的完整了。

那种火锅就像一部电影，有主演，有配角，绿叶配红花的才好看好吃。芋儿鸡、肥肠鸡、跳水蛙这个太具有本地特色的火锅在北京难觅踪迹，只能把馋虫先关进笼子里，每年回家一次性得到满足。另一些我恰好找到的，觉得这些可以推荐给大家。


1、【成都香倒拐耗儿鱼（北京店）】
地址：西三环北路82号（航天桥北侧）
人均：107元/人

成都香倒拐耗儿鱼

# 推荐理由：成都有没有这家店我是真不知道，不过作为一个重度鱼火锅爱好者可以负责的说，北京这家店是经过不少于20个四川人的检验的！火锅小料是正宗的原汤料，加了榨菜、炒黄豆、芹菜末，葱花香菜和豆豉，一下就很专业了有木有啊！耗儿鱼也是超大只，大概4条都有一斤的样子，两斤起点，所以一口吃下去很过瘾啊！


2、【超越火锅鸡】
地址：40元/人
人均：脑科医院红绿灯北边

# 推荐理由：长途跋涉坐绿皮去沧州只为吃火锅鸡，可以算得上是我吃货生涯中浓墨重彩的一笔了。半夜12点到，睡了饱饱一觉就迫不及待的连吃了两顿，对比连锁店，晚上路边寻找的小店更合我的胃口。
除了是用铜锅子以外，沧州火锅鸡的醋蒜蘸料也很独一无二，我有试过再吃四川火锅的时候配这个料，但怎么吃都感觉不对味，怪怪的，所以有机会一定要来沧州体验一下。

沧州有一条街都是卖火锅鸡的，生意看起来都很红火，每个店还有抽真空的外卖可以带走，可以说是很震撼了。但真相是这种美食出了沧州就销声匿迹，好像并没有影响周边的区域。这家是在沧州的地方贴吧上被推荐的，看起来非常简陋甚者很苍蝇馆子的火锅鸡，两斤足以，还配免费且不限量加的配菜，不论的钱包还是舌头都会感到超满足！


3、【一蛙面】
地址：东三环中路世桥国贸底商
人均：56元/人

# 推荐理由：虽然店名叫做一蛙面，但根据盆友们的反馈来看，牛蛙面并没有牛蛙煲好吃~
本周我也是亲测了中辣牛蛙煲，可以说是棒棒的了！里面配有牛蛙、鸡爪、青笋、土豆、年糕和生菜（？？？）闹不懂为啥要放那么多生菜，是为了充量么？？？除了生菜太辣没法吃以外，其他的都不错！量我也觉得很OK！另外一份牛蛙煲还配送一份葱油饼~葱油饼也不错，所有这家店有点川菜mix上海菜，很奇妙的感觉。


4、【重八牛府(小刀牛试店)】
地址：中关村南大街2号华宇时尚购物中心夹层(友谊宾馆对面)
人均：94元/人

# 推荐理由：我是看着这家店一点一点壮大的，从一个店现在开到了三个店，居然每个店每次都是排大队的节奏！最有特色的手打牛丸，非常弹牙，分生的熟的两种。锅底里会赠送熟的，吃着吃着，还有小哥哥端着肉盆子来现场为你下生牛丸，嗯~人少的话，其实牛丸都能吃到饱！还有他家的小料也超级有特色，不用自己调，老板自己研发了很多种口味，你可以把所有口味都要一遍，挨个品尝。我推荐秘制麻辣的~~

三、面食是个好东西

吃米长大的南方人到北方会面临被面食支配的恐惧，但是！其实只要是好吃的就完全没有在怕的啊~特别是在胃不好的时候，天冷的时候，饿炸了的时候，或者是没钱的时候，有一碗热腾腾的面或者粉简直就太美好了，一口上天堂，吃饱喝足也才几十块，真的是生活良心之选啊。

文邹邹的也可以说，精熬得好汤，配上弹、细、软、劲的面条，吸足了汤水，或者是浇上肉末，蒜末，油辣子，香辣十足，南方人的吃米主义也就被瞬间打到。

好吃又管饱的面食，我觉得下面的可以有！

1、【东方宫中国兰州牛肉面（交大东路店）】
地址：交大东路66号院钻河公馆105号商底（近西直门地铁A口）
人均：25元/人

# 推荐理由：对于一个没去过兰州的人来说，这家店在北京的兰州拉面领域，真的算得上是美食暴击了！是会久了不吃就超级想念的店。

我也是不明白，同样是拉面，好吃的和不好吃的怎么会差这么多。这家在我心中绝对是北京拉面TOP1。第一次去吃的时候，把汤都喝干了，要知道，人家可是一只萌妹子啊~

现在每次去我都会在大众点评上买一个肉蛋双飞单人套餐，有面有肉有蛋还有小菜，牛肉片很舍得给，然后面要三细，泡一泡，汤汁吸的足够，滋味最足！唉~写着写着就饿了。还有店里自制的熬茶也一定要喝哦，甜甜的~在碗柜边上自取。


2、【我没在点评上找到这家店，但是还是想放上来，改天去实地拍！】
地址：
人均：牛肉8块，猪肉7块！

# 推荐理由：四川人才懂得的锅盔！！！老板说是从新都军屯学来的手艺，可以说是很正宗了，非常有我小时候的味道。锅盔有猪肉馅和牛肉馅两种，是揉在面里面的那种。新出锅烫嘴的不行，问着就特别香，咬起来酥酥脆脆的真的是忍不住吹着起吃完它。要说缺点，就是有点油了....

从小窗口买就可以！店里其他的产品我没有吃过就不予置评啦。

3、【北27號●廿七禾尚】
地址：三里屯那里花园一层北侧
人均：53元/人

# 推荐理由：超级有B格和小清新的一家店，但绝不是心絮其外败絮其中的那种网红店。地点在一个小花园里，第一次去的时候真的是觉得打开了新世界啊，主打的是西北菜，灰豆子，甜坯子，酿皮和拌面！怎么形容呢~如果在屯儿里想试试不一样的，指路这家！

4、【四眼仔车仔面】
地址：三里屯路33号33大厦5层5027商铺
人均：69元/人

# 推荐理由：我没有在广东吃过真正的车仔面，但这家觉得是我在北京吃的最好吃的一家，比小老虎好吃一万倍！点菜的方式也很像1950年代的香港，那时候贩卖车仔面的木头车中放置金属造的“煮食格”，分别装有汤汁、面条和配料，顾客可自由选择面条，配料和汤汁，通常十多块钱就可饱吃一顿。唯二的差别是，搬进了餐厅和十多块钱是吃不饱的~

四、恐甜患者的甜品推荐

我其实不太爱吃甜食，从口味上来看，是彪莽大汉，非常的不少女。所以甜品店的推荐，大家看看就好！不过这几家应该都比较出名了~~~也算是经得起嘴巴的检测啦！而且我去过的也不算很多，因为相比这个，我更喜欢往苍蝇馆子里钻。不过吃过的几家都不错，主要归功于师傅带的好！哈哈哈哈~

1、【DA GIULIANO】
地址：三里屯北小街1号(一坐一忘丽江主题餐北面)
人均：43元/人

# 推荐理由：位置在使馆区，如果是一个阳光惬意的午后，在这里度过总不会错！各种各样的小甜点，7块钱一块，甜是真的很甜，不过配上红茶就能解腻，也有很多外国食客，看起来会是逼格满满，不过最好玩的一点是付款的时候，你发现这家中文店名其实特别接地气，叫老朱饼店，非常有杂货小铺子的即视感，跟眼前的画面很有反差萌！哈哈哈哈哈哈！


2、【元古本店】
地址： 箭厂胡同34号
人均：82元/人

# 推荐理由：中式的甜品店，还挺有禅意的，会根据节气的不同出不同的品类，食材也是用的最应季的那种，是我吃过的独两份（另一家观品，因为胡同整改暂停营业了）。环境也很雅致，果然只有广阔的空间才配得上新中式啊，没有花花绿绿的装饰和动次打次，关键是所有的甜品都不甜，口味刚刚好，能吃出食材本质的味道，对于我来说，还不错啦~
